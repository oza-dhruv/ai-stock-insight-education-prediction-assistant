# AI Powered Stock Insight, Education, and Prediction Assistant

An AI-powered platform designed to help beginner investors understand stocks through data analysis, technical indicators, sentiment analysis, short-term trend prediction, and AI-generated explanations.

## Features

- Stock ticker input
- Company and stock data retrieval
- Technical indicators (SMA, EMA, RSI, MACD)
- Historical stock visualizations
- Bullish / bearish signal summary
- News sentiment analysis
- Short-term trend prediction
- LLM-based beginner-friendly explanations

## Tech Stack

Python  
Streamlit  
Pandas  
NumPy  
yfinance  
Plotly  
Matplotlib  
Scikit-learn  
OpenAI API

## Project Goal

Provide a beginner-friendly AI assistant that explains stock insights clearly without giving financial advice.
